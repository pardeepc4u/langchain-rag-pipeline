# LangChain RAG Pipeline

A Retrieval-Augmented Generation system built with LangChain and Python.

## Description

This repository demonstrates a simple implementation of the RAG pipeline using LangChain. The pipeline ingests documents, builds vector embeddings, and answers questions with source citations.

## Requirements

*   Python 3.8+
*   langchain
*   faiss
*   openai

## Usage

1.  Clone the repository: `git clone https://github.com/your-username/langchain-rag-pipeline.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the app: `python src/app.py`

## Example Use Cases

*   Answering questions with source citations
*   Document retrieval and summarization
